EntreNos – Privacy Policy

Last updated: 2025

Versión en Español

1. Información General

EntreNos es una aplicación de mensajería privada diseñada para permitir la comunicación entre personas. Esta Política de Privacidad explica cómo recopilamos, usamos y protegemos la información de los usuarios.

2. Información que recopilamos

  • Nombre y apellido
  • Correo electrónico
  • Fecha de nacimiento
  • Número de teléfono
  • Contenido de mensajes enviados dentro de la aplicación
  • Imágenes compartidas
  • Token de notificaciones (Firebase Cloud Messaging)

3. Permisos del dispositivo

  • Cámara: para enviar imágenes dentro del chat.
  • Almacenamiento: para seleccionar o guardar imágenes.
  • Notificaciones: para informar sobre nuevos mensajes.

4. Cómo usamos la información

La información se utiliza exclusivamente para:

  • Operar la aplicación.
  • Permitir la comunicación entre usuarios.
  • Enviar notificaciones relacionadas con la actividad del chat.
  • Garantizar seguridad y funcionamiento adecuado.

5. Servicios de terceros

EntreNos utiliza servicios de Google Firebase para autenticación, almacenamiento y notificaciones. Estos servicios pueden procesar datos conforme a sus propias políticas de privacidad.

6. Menores de edad

La aplicación puede ser utilizada por menores. Solo recopilamos la información estrictamente necesaria para operar el servicio.

7. Retención de datos

Los datos se almacenan mientras el usuario mantenga su cuenta activa. El usuario puede solicitar eliminación de su cuenta y datos asociados.

8. Seguridad

Implementamos medidas razonables para proteger la información contra accesos no autorizados.

9. Cambios en esta política

Podemos actualizar esta Política de Privacidad periódicamente. Los cambios serán publicados en esta misma página.

10. Contacto

Para consultas relacionadas con privacidad:
contacto@entrenosapp.com

English Version

1. General Information

EntreNos is a private messaging application designed to enable communication between users. This Privacy Policy explains how we collect, use, and protect user information.

2. Information We Collect

  • First and last name
  • Email address
  • Date of birth
  • Phone number
  • Message content within the app
  • Shared images
  • Notification token (Firebase Cloud Messaging)

3. Device Permissions

  • Camera: to send images in chat.
  • Storage: to select or save images.
  • Notifications: to inform about new messages.

4. How We Use Information

Information is used solely to:

  • Operate the application.
  • Enable communication between users.
  • Send chat-related notifications.
  • Ensure proper functionality and security.

5. Third-Party Services

EntreNos uses Google Firebase services for authentication, storage, and notifications. These services may process data according to their own privacy policies.

6. Minors

The application may be used by minors. We only collect data strictly necessary to operate the service.

7. Data Retention

Data is stored while the user account remains active. Users may request account and data deletion.

8. Security

We implement reasonable security measures to protect user information from unauthorized access.

9. Changes to This Policy

We may update this Privacy Policy periodically. Updates will be published on this page.

10. Contact

For privacy-related inquiries:
contacto@entrenosapp.com
